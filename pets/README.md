# Картинки питомцев — Люся 🐱 и Рада 🐶

Положи сюда картинки, и они автоматически появятся на курсе вместо рисованных.

## Обязательные файлы
- `lusya.png` — кошечка Люся
- `rada.png` — собачка Рада

## Необязательные (мордочка меняется на ответ)
- `lusya-happy.png`, `lusya-sad.png`
- `rada-happy.png`, `rada-sad.png`

## Требования к картинкам
- Формат **PNG с прозрачным фоном** (без белого/цветного фона).
- Квадрат, питомец по центру, смотрит на зрителя.
- Размер ~512×512 px, файл лучше < 300 КБ.
- Стиль — мультяшный/мягкий. Можно нарисовать в GPT по реальным фото.

Если файла нет — показывается аккуратный векторный рисунок (его цвета можно
поправить в `css/extra.css`, секции «Люся»/«Рада»).
