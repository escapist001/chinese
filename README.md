# 中文 · Тренажёр китайского языка (HSK 1)

Веб-приложение для изучения китайского с нуля до полного HSK 1: 150 слов, озвучка, 12 режимов тренировок и питомец, который болеет за тебя.

**[▶ Открыть живую версию](https://escapist001.github.io/chinese/)** — работает в браузере, ставится как приложение, не требует интернета после первой загрузки.

## 💡 Что это и зачем

Проект начинался как подарок близкому человеку, который учит китайский. Сначала это была простая страничка с карточками. Потом захотелось добавить звук, потом прогресс, потом чтобы работало в метро без связи. В итоге получилось полноценное PWA с полным словарём HSK 1 и геймификацией.

Задача у приложения одна: чтобы заниматься было не скучно и было понятно, куда двигаться дальше. Отсюда структура из юнитов и уроков, а не бесконечная лента слов.

## ✨ Возможности

**Обучение**
- Полный словарь HSK 1: 10 юнитов, 30 уроков, 150 слов
- 405 аудиофайлов с нативным произношением
- 12 режимов тренировок: карточки, аудирование, письмо и другие
- Этимология иероглифов — откуда взялся знак и почему он так выглядит

**Геймификация**
- Питомец-компаньон (кошка или собачка на выбор), реагирует на успехи словами поддержки
- «Сад знаний» — визуальный прогресс изучения
- Печенье с предсказанием
- Генератор китайского имени

**Интерфейс**
- Плавные анимации, смена темы с эффектом лепестков
- Адаптив под мобильные — основной сценарий использования как раз телефон

## 🛠 Стек

- Ванильный JavaScript, без фреймворков (никаких React/Vue)
- HTML + CSS
- PWA: service worker, manifest.json
- Хостинг: GitHub Pages (статика + `.nojekyll`)

## 🔩 Что интересного под капотом

- **Модульная ваниль.** Код разбит на `js/core`, `js/data`, `js/views`, плюс отдельные `app.js` и `components.js`. Без сборщика и фреймворка это дисциплинирует: границы между слоями держатся на договорённостях, а не на инструментах.
- **Офлайн-first.** Service worker (`sw.js`) со стратегией network-first: онлайн отдаёт свежую версию, офлайн — кэш. Приложение полностью работает без сети.
- **Версионирование кэша.** При деплое новой версии старый кэш вычищается, пользователь не застревает на устаревших файлах — на статическом хостинге это единственный способ управлять обновлениями.
- **405 аудиофайлов.** Вся озвучка — реальные mp3 с нативным произношением, а не синтез в браузере. Организовать, подключить и раздавать такой объём статики оказалось отдельной задачей.

## 🚀 Запуск локально

Это статический сайт, сборка не нужна.

```bash
git clone https://github.com/escapist001/chinese.git
cd chinese
python -m http.server 8000
```

Открыть `http://localhost:8000`. Подойдёт любой статический сервер; можно и просто открыть `index.html`, но service worker заработает только по http.
