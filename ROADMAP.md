# 中文 — Большой роадмап развития курса

План превращения тренажёра в **профессиональную языковую платформу** (ориентир — Duolingo / HelloChinese / Skritter + методика HSK).
**Принцип: ноль затрат.** Только бесплатные технологии, без платных API, без аккаунтов с деньгами.
Стек прежний: статический сайт (HTML/CSS/JS), весь прогресс в браузере (localStorage), хостинг GitHub Pages.

**Устройство Жени — iPhone.** Учитываем: на iOS все браузеры = Safari. Распознавание речи там капризное → опираемся на
визуализатор тонов (работает с микрофоном офлайн) и заранее записанное аудио.

Обозначения: **Польза** (для Жени-ученицы) · **Портфолио** (вес для резюме) · **Сложн.** ◔ малая · ◑ средняя · ● крупная.

## 🎬 Сквозная система анимаций (вплетаем во все фазы)
Даня фанатеет по красивым анимациям → делаем их фирменной чертой. Принцип: **в тему (тушь/кисть/печать),
оркестрованные ключевые моменты, а не хаос**. Всё бесплатно (CSS + чуть JS/canvas). Уважаем `prefers-reduced-motion`,
анимируем только `transform`/`opacity` (60 fps).

**Геро и приветствие:**
- 🖌️ **Самопись кистью** — 你好 «рисуется» штрих за штрихом в клетке 田字格 при загрузке (Hanzi Writer). Главный on-brand эффект.
- 💧 **Расплыв туши** — мягкое пятно 水墨 проявляется за героем (SVG/CSS).
- 🔢 **Счётчики набегают** — XP, слова, точность плавно «крутятся» до значения.

**Переходы между экранами:**
- 🪶 **Кисть-вайп** — мазок кистью сметает один экран и открывает другой (View Transitions API + запасной фейд).
- 📜 **Разворот свитка** — для крупных переходов/уровней контент «разворачивается» как свиток.

**Микро-взаимодействия:**
- ✅ **Галочка рисуется** при верном ответе (анимация SVG-пути); ❌ бумага «вздрагивает» при ошибке.
- 🎴 пружинный flip карточек, лёгкий 3D-тилт, чернильная «рябь» на нажатии кнопок.
- 📊 кольца/полосы прогресса плавно заполняются; ховеры с подрастающим подчёркиванием.

**Моменты-награды (оркестрованные):**
- 🏮 **Печать 印章 впечатывается** — штамп «бьёт» с лёгким наклоном, дрожью и расплывом чернил (улучшаем `seal-pop`).
- 🎉 **Чернильные брызги** — capли-конфетти на canvas при завершении урока/квиза (бесплатно, без библиотек).
- 🆙 **Level-up** — свиток разворачивается и показывает новое звание.

**Учебные анимации:**
- ✍️ кисть тянет след в режиме письма; 〰️ кривая тона «прочерчивается»; 📈 живая линия высоты голоса в визуализаторе тонов.

**Фоновая атмосфера (очень деликатно):**
- иероглифы-водяные знаки чуть плывут при скролле (параллакс); медленный дрейф чернильных пятен.

> ⚠️ iPhone: View Transitions API заводится на свежем Safari, для старых — плавный фейд-фолбэк; canvas-брызги и CSS работают везде.

## 🧭 Рекомендованный маршрут (мой совет «куда идём»)

**Этап 1 — Голос и звук** *(Фаза 1):* нейро-озвучка MP3 + визуализатор тонов.
→ Сразу закрывает твою главную просьбу, и это слышно/видно с первого открытия.

**Этап 2 — Навыки** *(Фаза 2):* письмо от руки + тренажёр тонов + работа над ошибками.
→ Проект становится «настоящим тренажёром», а не карточками.

**Этап 3 — Разнообразие + структура** *(Фазы 3–4):* пара новых режимов + привязка к HSK + цели уроков.
→ Появляется «программа курса», заниматься не скучно.

**Этап 4 — Вау-визуал** *(Фаза 5):* стена печатей + календарь активности + графики.
→ Именно это впечатляет на скриншотах в портфолио.

**Этап 5 — Под Женю** *(Фаза 6):* избранное + тёплые сюрпризы + работа над слабыми словами.

**Этап 6 — Упаковка** *(Фаза 7):* PWA (офлайн на айфоне) + мобильное меню.
→ Финальный лоск: «настоящее приложение».

**Параллельно:** по чуть-чуть пополняем словарь и темы из backlog'а.

> Логика: сначала **то, что просил (звук/голос)** → **польза для ученицы** → **разнообразие и методика** →
> **внешний эффект** → **персонализация** → **упаковка**. На каждом шаге сайт остаётся целым и рабочим.

## ⭐️ ОБЯЗАТЕЛЬНО В КОНЦЕ
**Полная адаптация под мобильные** (особенно iPhone Жени): нижняя навигация, проверка всех экранов на 375px,
крупные тап-цели, отступы под «чёлку»/жесты. Делаем финальным этапом после контента. (Просьба Дани, 2026-06-28.)

## ✅ Зафиксированные решения
- Бюджет: **0 ₽**. Только бесплатные технологии.
- Озвучка: **нейро-MP3 (edge-tts)**, генерим заранее, кладём в репо. Системный TTS — запасной.
- Голос Жени: **визуализатор тонов** (надёжно на iPhone) + распознавание «по возможности».
- «Настоящая оценка произношения как в платных» (Azure и т.п.) — **не делаем** (нужен платный/серверный ключ).
